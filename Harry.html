AI in Natural Language Processing

  • Natural language processing (NLP) is a field of artificial intelligence where computational methods are used analyze and interpret human language. NLP encompasses all algorithms that process natural language, some of these algorithms include:
  • Automatic Speech Recognition (ASR) converts spoken audio from the user into a text transcription
  • Natural Language Understanding (NLU) takes the transcription of what the user said and predicts their intention in a way that makes sense to the program
  • Text to Speech (TTS) is the algorithm responsible for synthesising the output voice of the assistant.
  • The most wide spread application of NLP is virtual assistants, sometimes referred to as intelligent virtual assistants use the techniques outlines in natural language processing to allow the user to interface with the computer using spoken language. The language is then translated and interpreted as a command that can be understood by the software which is then executed. This greatly improves the accessibility of computer systems

  • Above: Google assistant is one of the most widely used virtual assistants found in modern day smartphones
  • As the technology improves, AI virtual assistants are thought to become more contextual. New machine learning algorithms such as GPT-2, which has been trained on over 8 million web pages, can 'remember' data from previous commands inputted by the user and real world knowledge
